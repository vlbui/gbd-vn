# raw_data/air_pollution/

IHME air pollution exposure + relative-risk files, used by papers
`11_pm25_ncd` and `12_heat_attributable`. Original download page:
<https://ghdx.healthdata.org/record/ihme-data/gbd-2023-air-pollution-exposure-estimates-1990-2023>

## exposure/

| File | Contents |
|------|----------|
| `IHME_GBD_2023_AIR_POLLUTION_1990_2023_PM.zip` | Annual mean PM2.5 gridded exposure + population-weighted country means |
| `IHME_GBD_2023_AIR_POLLUTION_1990_2023_HAP.zip` | Household air pollution prevalence + exposure |
| `IHME_GBD_2023_AIR_POLLUTION_1990_2023_NO2_Y20251010.csv` | Annual mean NO2 surface concentration (v20251010) |
| `IHME_GBD_2021_AIR_POLLUTION_1990_2021_OZONE.zip` | Seasonal ozone exposure (GBD 2021 round — not yet refreshed for 2023) |

## rr_curves/

| File | Contents |
|------|----------|
| `IHME_GBD_2023_AIR_POLLUTION_1990_2021_RR_CURVES.zip` | Cause-specific relative-risk curves for PM2.5, HAP, NO2, ozone |
